📶 Offline & Hintergrund
Offline: Die App funktioniert vollständig ohne Internet-Verbindung – alle Assets sind gecacht.

Hintergrund-Benachrichtigungen: Funktionieren solange der Browser läuft (auch bei gesperrtem Bildschirm). Wenn der Browser komplett geschlossen ist, können keine Benachrichtigungen gesendet werden – das ist eine Browser-Beschränkung ohne Server-Push.
📱 Als App installieren (APK)
1. Diese App auf GitHub Pages hosten
2. pwabuilder.com öffnen → URL eingeben → Android APK generieren
3. APK auf Android installieren (Sideloading aktivieren)

TWA-kompatibel: manifest.json ist vollständig für Trusted Web Activity optimiert.
